# CRM 與 Admin 功能重構計劃

## 功能分類

### 需要移到 Admin 頁面（前端控制功能）：
1. **公告管理** (announcements)
   - 控制前端網站顯示的公告
   - 包含：公告列表、新增/編輯/刪除、圖片上傳

2. **服務項目** (services)
   - 控制前端網站和 LINE 顯示的服務
   - 包含：服務列表、新增/編輯/刪除、圖片上傳、拖拽排序

3. **分店設定** (stores)
   - 控制前端網站和 LINE 顯示的分店
   - 包含：分店列表、新增/編輯/刪除、營業時間設定

### 保留在 CRM 頁面（LINE 相關功能）：
1. **用戶管理** (users) - LINE 用戶管理
2. **再行銷設定** (remarketing) - LINE 再行銷
3. **關鍵字管理** (keywords) - LINE 機器人關鍵字
4. **機器人設定** (settings) - LINE 機器人設定
5. **預約管理** (bookings) - LINE 預約管理

## 重構步驟

1. ✅ 分析現有功能分類
2. ⏳ 將公告管理功能移到 Admin
3. ⏳ 將服務項目管理移到 Admin
4. ⏳ 將分店設定移到 Admin
5. ⏳ 從 CRM 移除相關標籤頁和代碼
6. ⏳ 測試功能是否正常
