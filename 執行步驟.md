# 執行 Announcements Migration 步驟

## ⚠️ 重要：請複製 SQL 檔案，不是 Markdown 檔案！

## 步驟 1：選擇正確的檔案

請使用以下 SQL 檔案（`.sql` 結尾）：
- ✅ `ANNOUNCEMENTS_MIGRATION_COMPLETE.sql` ← **推薦使用這個**
- 或 `ANNOUNCEMENTS_MIGRATION_FINAL.sql`

## 步驟 2：複製 SQL 內容

1. 打開 `ANNOUNCEMENTS_MIGRATION_COMPLETE.sql` 檔案
2. **全選**所有內容（Cmd+A 或 Ctrl+A）
3. **複製**（Cmd+C 或 Ctrl+C）

## 步驟 3：在 Supabase SQL Editor 執行

1. 前往 Supabase Dashboard
2. 點擊左側選單的 **SQL Editor**
3. 點擊 **New query**
4. **貼上**剛才複製的 SQL 內容（Cmd+V 或 Ctrl+V）
5. 點擊 **Run** 或按 **Cmd+Enter**（Mac）或 **Ctrl+Enter**（Windows）

## ⚠️ 常見錯誤

- ❌ **錯誤**：複製 `SQL_REVIEW_SUMMARY.md`（Markdown 檔案）
- ✅ **正確**：複製 `ANNOUNCEMENTS_MIGRATION_COMPLETE.sql`（SQL 檔案）

## 檔案對比

### SQL 檔案（✅ 使用這個）
- 檔案名稱：`ANNOUNCEMENTS_MIGRATION_COMPLETE.sql`
- 開頭：`-- ============================================`
- 包含：SQL 語句（CREATE TABLE, CREATE FUNCTION 等）

### Markdown 檔案（❌ 不要使用）
- 檔案名稱：`SQL_REVIEW_SUMMARY.md`
- 開頭：`# SQL 檔案檢查總結`
- 包含：Markdown 格式的文字說明
